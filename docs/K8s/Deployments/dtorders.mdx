# DT-Orders

## Quickstart

Download the dtorders application manifest.

```bash
curl https://suchcodewow.io/manifests/dtorders/dtorders-all.yaml -L -o dtorders-all.yaml
```

Replace `dev.local` in this yaml file with the IP address of your nginx load balancer.

```bash
sed -i "s/dev.local/$(kubectl get svc ingress-nginx-controller -n ingress-nginx -o=custom-columns=EXTERNAL-IP:.status.loadBalancer.ingress[0].ip --no-headers)/g" dtorders-all.yaml
```

Apply the updated yaml file.

```bash
kubectl apply -f dtorders-all.yaml
```
