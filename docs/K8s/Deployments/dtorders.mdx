# DT-Orders

## Quickstart

```bash
kubectl apply -f https://suchcodewow.io/manifests/dtorders/deployall.yaml
```

Get the load balanced IP of your cluster.

```bash
kubectl get svc ingress-nginx-controller -n ingress-nginx -o=custom-columns=EXTERNAL-IP:.status.loadBalancer.ingress[0].ip --no-headers
```

### command reference

attach to a running pod

kubectl exec --stdin --tty shell-demo -- /bin/bash
