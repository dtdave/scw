# Dynatrace Install

These steps will install the Dynatrace operator in your environment.

## Create and Deploy Manifest

- In your Dynatrace tenant, click _Deploy Dynatrace_ from the left navigation and then _start installation_.

- Select _Kubernetes_.

- Enter name in **Name**.

- Click _create token_ on the **Operator Token** line.

- Switch _Skip SSL certificate check_ to **on**.

- Click **Download dynakube.yaml**.

- Copy the dynakube.yaml file to your terminal window.

- Copy and paste the commands shown in Dynatrace tenant window.
